Overview


Gloomy Day Clean-Up is a simulation game that involves organizing a messy room based on instructions displayed on the screen. The goal is for players to complete the tasks assigned to them, resulting in a neat and organized room.

Process


Planning and Proposal

Initially, I planned a proposal of the concept explaining the project idea, user interaction, main process and a storyboard.

Concept Proposal

Drawings and Animation

To create the simulation, I used Sketchbook to draw each component while referring to internet images for color schemes. I designed various animations, including spiders moving and changing weather featuring moving rain and clouds.

Drawings created in Sketchbook

Development

Regarding the code, I used packages to organize it and created various states. I have a total of 19 states using the FSM model for the interactions, sounds and various animations. I utilized perlin noise to create the dust object, the factory pattern to help instantiate objects and arraylists, and the decorator pattern to construct painting objects for GUI widget utilization.

This is a sample code. It demonstrates the use of the factory pattern to initialize objects in Java

Audio

The majority of the audio was recorded through my phone's voice memos app, while I also obtained additional sounds from external sources such as Fesilyan Studios.

Audio from Voice Memos Application

Challenges


Issues

While working on the project, I faced issues with transitioning states as some of them were not functioning properly, and also encountered problems when implementing arraylists with the factory pattern.

Resolution

To overcome this issue, I carefully reviewed my code and identified small mistakes such as case-sensitivity in strings. For the factory pattern, I experimented with different versions of the code until I found a solution and noted down the reasons for my past unsuccessful attempts.

Reflection


What I've Learned

By participating in this project, I gained knowledge about basic knowledge of Java programming language including concepts such as inheritance, pull-up operations and polymorphism.

Areas of Improvement

Upon reflecting on the project, I wish to include more animations and interactions to make the gameplay longer and more enjoyable.
