Typefaces

The main font is Lato by Łukasz Dziedzic. Lato not only exudes a friendly and inviting aura, but also conveys a sense of professionalism.




Interactive Items

In-text link

.in-text-link

Sample code:

<a href="#">Click here to visit our website!</a>

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>
